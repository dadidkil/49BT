# Discord Bot 49 Battalion

Многофункциональный Discord бот для сервера 49 Battalion с функциями верификации, управления голосовыми каналами и воспроизведения музыки.

## Функции

- 🔐 Система верификации с капчей
- 🎵 Воспроизведение музыки из YouTube
- 🎙️ Управление приватными голосовыми каналами
- 👋 Автоматические приветствия
- ⚙️ Административные команды

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/discord-bot-49bt.git
cd discord-bot-49bt
```

2. Создайте виртуальное окружение и активируйте его:
```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# Linux/Mac
source .venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` в корневой директории и добавьте токен бота:
```
DISCORD_TOKEN=ваш_токен_бота
```

5. Установите FFmpeg для работы с аудио:
- Windows: Скачайте с [официального сайта](https://ffmpeg.org/download.html) и добавьте в PATH
- Linux: `sudo apt-get install ffmpeg`
- Mac: `brew install ffmpeg`

## Настройка

1. Пригласите бота на сервер с необходимыми правами:
   - Управление каналами
   - Управление ролями
   - Отправка сообщений
   - Подключение к голосовым каналам
   - Воспроизведение звука

2. Настройте бота с помощью команд администратора:
   - `/set_verification_channel` - установить канал верификации
   - `/set_member_role` - установить роль участника
   - `/set_welcome_channel` - установить канал приветствий
   - `/set_private_vc_category` - установить категорию для приватных каналов
   - `/set_private_vc_lobby` - установить канал-лобби

## Использование

### Команды для всех пользователей

- `/private_panel` - открыть панель управления приватным каналом

### Команды администратора

- `/set_verification_channel` - установить канал верификации
- `/set_member_role` - установить роль участника
- `/set_welcome_channel` - установить канал приветствий
- `/set_private_vc_category` - установить категорию для приватных каналов
- `/set_private_vc_lobby` - установить канал-лобби
- `/ping` - проверить работоспособность бота

## Логирование

Бот ведет логи в файл `bot.log` и выводит их в консоль. Логи содержат информацию о:
- Запуске и остановке бота
- Ошибках и исключениях
- Действиях пользователей
- Изменениях в конфигурации

## Безопасность

- Токен бота хранится в файле `.env`
- Все команды администратора требуют соответствующих прав
- Верификация защищена от ботов с помощью капчи
- Приватные каналы доступны только их владельцам

## Поддержка

При возникновении проблем:
1. Проверьте логи в файле `bot.log`
2. Убедитесь, что все зависимости установлены
3. Проверьте права бота на сервере
4. Создайте issue в репозитории проекта
